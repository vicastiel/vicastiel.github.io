Transformation de Fourier (FFT)

La transformation de Fourier est une technique math√©matique puissante et normalement complexe a mettre en oeuvre. Heureusement scipy.fftpack rend cette technique tres simple a impl√©menter

La transformation de Fourier permet d'analyser les fr√©quences qui composent un signal p√©riodique (qui se r√©pete avec le temps). Cette op√©ration produit un graphique que l'on appelle Spectre.

Une fois le Spectre g√©n√©r√©, il est possible de filtrer les bruits ind√©sirables, ou bien de s√©lectionner seulement certaines fr√©quences, ou d'en att√©nuer d'autres... les possibilit√©s sont infinies.

Dans l'exemple ci-dessous, nous voyons comment filtrer un signal noy√© dans du bruit.

# Cr√©ation d'un signal p√©riodique noy√© dans du bruit.

from scipy import fftpack

# cr√©ation des variables Fourier et Fr√©quences, qui permettent de construire le spectre du signal.

# filtre du spectre avec du boolean indexing de Numpy

# Visualisation du spetre propre

# Transformation de Fourier Inverse: genere un nouveau signal temporel depuis le spectre filtr√©
filtered_signal = fftpack.ifft(fourier)

# Visualisation des r√©sultats

data = bitcoin.copy()
data['Buy'] = np.zeros(len(data))
data['Sell'] = np.zeros(len(data))

data['RollingMax'] = data['Close'].shift(1).rolling(window=28).max()
data['RollingMin'] = data['Close'].shift(1).rolling(window=28).min()
data.loc[data['RollingMax'] < data['Close'], 'Buy'] = 1
data.loc[data['RollingMin'] > data['Close'], 'Sell'] = -1

def plot_decision_boundary(clf, X, y, axes=[-1.5, 2.45, -1, 1.5], alpha=0.5, contour=True):
    
    h = .02
    x_min, x_max = X[:, 0].min() - .5, X[:, 0].max() + .5
    y_min, y_max = X[:, 1].min() - .5, X[:, 1].max() + .5
    xx, yy = np.meshgrid(np.arange(x_min, x_max, h),
                         np.arange(y_min, y_max, h))
    
    
    Z = clf.predict_proba(np.c_[xx.ravel(), yy.ravel()])[:, 1]
    Z = Z.reshape(xx.shape)
    plt.contourf(xx, yy, Z, alpha=.5)
    
    plt.scatter(X[:,0], X[:,1], c=y, alpha=0.8, edgecolors='k')
